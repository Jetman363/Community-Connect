# BlueCore — Frontend Dashboard

Next.js command center prototype for the law enforcement operations platform.

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Demo Login

Use the pre-filled credentials on the login page, or any agency email with a password of 4+ characters.

## Pages

| Route | Description |
|-------|-------------|
| `/login` | Secure agency sign-in |
| `/dashboard` | Command center with live incidents, units, alerts, trends |
| `/reports/ai` | AI report writing with human-in-the-loop approval |
| `/investigations` | Investigations workspace with link analysis |
| `/rms/search` | RMS report search and detail view |
| `/profile` | Officer profile and activity |
| `/admin/settings` | Agency, security, AI, and integration settings |

## Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React icons

## Design

GovTech command center aesthetic — dark theme, high information density, status-driven color coding, and professional data tables suitable for municipal through federal agency operations.
